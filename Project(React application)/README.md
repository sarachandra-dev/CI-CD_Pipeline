# QuickShare - it is an quick data transfer site

-Share text securely with a 6-character code.

## Setup

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
### How to run
```bash
open two terminals
1.first run : node server.js
2.second run : npm run dev
```
### Note: navigate to the backend and frontend should be done before running the command


## Usage
- Visit `http://localhost:5000`
- **Share**: Paste text → Generate Code → Share the 6-char code
- **Retrieve**: Enter the code → Get the text back
